## 代码中使用

```python
boundary_A = np.array([32.90, 38.61, ...])  # F%
boundary_B = np.array([62.10, 53.38, ...])  # A%
boundary_C = np.array([5.00, 8.00, ...])    # M%
# ternary_to_xy(top=F, left=A, right=M)
b_x, b_y = ternary_to_xy(boundary_A, boundary_B, boundary_C)
ax.plot(b_x, b_y, 'k-', lw=1.8, zorder=5)
```

## 区域填充

- Tholeiitic（上方区域）：从 F 顶角 → A 左下角 → M 右下角 → 分界线反向
- Calc-Alkaline（下方区域）：从 F 顶角 → 分界线 → M 右下角
