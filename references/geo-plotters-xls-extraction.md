## 背景

用户通过 Excel 插件 **GeoPlotters** 生成 Winchster-Floyd (1977) 分类图底图，保存为 XLS 文件。该 XLS 包含两个 sheet：
- **Chart**：嵌入式 Excel 图表（XY 散点图 + 文字标注 "GeoPlotters"）
- **Plotting parameters**：100 行 x 2 列（Nb/Y ppm, Zr/Ti ppm），即底图分界线的数字化点

### 图表分析

- **图表类型**：XY Scatter（散点图），单一系列连接 100 个数据点
- **X 轴**：Nb/Y (ppm) — 对数刻度
- **Y 轴**：Zr/TiO₂ (ppm) — 对数刻度  
- **线色**：#969696（灰色），实线
- **数据源**：`=Plotting parameters!$A$2:$B$101`
- **系列名称**："Plotting parameters"
- **无标记点、无填充**

### OLE 嵌入内容

XLS 文件中的 Chart sheet 在 BIFF8 二进制中嵌入了 OOXML DrawingML 资源包。用以下方法提取：

```python
import zipfile, io

with open('/tmp/wf.xls', 'rb') as f:
    data = f.read()

# 找到 ZIP 签名 PK\x03\x04
for i in range(len(data) - 4):
    if data[i:i+4] == b'PK\x03\x04':
        zip_start = i
        break

with zipfile.ZipFile(io.BytesIO(data[zip_start:])) as zf:
    for name in zf.namelist():
        print(name, zf.getinfo(name).file_size)
```

嵌入的 ZIP 包含 4 个文件：
- `[Content_Types].xml`
- `_rels/.rels`
- `drs/shapexml.xml` — 指向 GeoPlotters 图标/文本框
- `drs/downrev.xml` — 向下兼容数据

## 分界线分段方法

100 个数据点按顺序描摹完整图版轮廓，通过以下方式分段为独立边界曲线：

1. 计算相邻点在 log 空间中的欧几里得距离
2. 距离 > 0.4 处视为分段边界（不同分界线之间的跳跃）
3. 使用 XLS 中的精确坐标（6-7 位小数精度），而非手动复制的粗略值

```python
import numpy as np

# 数据点
pts = np.array([(nb_y[i], zr_ti[i]) for i in range(100)])
log_pts = np.log10(pts)
dists = np.sqrt(np.sum(np.diff(log_pts, axis=0)**2, axis=1))

# 分段
segments = []
start = 0
for i, d in enumerate(dists):
    if d > 0.4:
        segments.append((start, i))
        start = i + 1
segments.append((start, len(pts)-1))
```
