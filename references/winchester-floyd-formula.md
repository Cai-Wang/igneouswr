# Winchester & Floyd (1977) Zr/TiO₂ vs Nb/Y 分类图公式推导

## 原文献公式

Winchester & Floyd (1977) 定义的纵轴为：

**Zr / TiO₂ (×0.0001)**

其中两个变量都是 **ppm** 单位级别。

## 数据库单位差异

地球化学数据库中常见：

| 变量 | 文献单位 | 数据库常见单位 |
|------|----------|---------------|
| Zr | ppm | ppm（一致） |
| TiO₂ | ppm | **wt%（重量百分比）** |

## 换算推导

TiO₂ 数据库中是 wt%，要转 ppm 需乘以 10000：

```
TiO₂(ppm) = TiO₂(wt%) × 10000
```

代入文献公式：

```
Zr(ppm) / TiO₂(ppm) = Zr(ppm) / (TiO₂(wt%) × 10000)
                    = Zr(ppm) / TiO₂(wt%) / 10000
                    = (Zr / TiO₂_wt%) × 0.0001
```

## 代码实现

```python
# _classification.py:297
zr_tio2[mask_zr] = zr[mask_zr] / tio2[mask_zr] * 0.0001
```

## 典型值验证

| 样品类型 | Zr (ppm) | TiO₂ (wt%) | 计算结果 (×0.0001) | 原图范围 |
|----------|----------|------------|-------------------|---------|
| 玄武岩 | ~100 | ~1.0 | 0.01 | ✓ 落在 Subalkaline Basalt 区 |
| 安山岩 | ~150 | ~0.8 | 0.01875 | ✓ 落在 Andesite 区 |
| 流纹岩 | ~300 | ~0.2 | 0.15 | ✓ 落在 Rhyolite 区 |
| 碱性玄武岩 | ~200 | ~2.0 | 0.01 | ✓ 落在 Alkali Basalt 区 |

## 常见错误

- **错误**：直接 `zr / tio2_wt%`（缺 ×0.0001），得到 100~300 级的值，远超出图范围 (0.001~1)
- **混淆源**：有时文献中的 `TiO₂` 直接标注为 "TiO₂ (ppm)" 或 "TiO₂ (wt%)"，需确认数据源的实际单位

## 配合的横轴

Nb/Y：两者数据库中都为 ppm，直接相除即可，无需换算。

## 纵轴标签

推荐标注为 `Zr/TiO₂ (×0.0001)` 以明确公式含换算因子。
